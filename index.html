What are semantic elements?

Semantic elements are elements that provides clear meaning to both the browser and the developer.

Semantic elements can be grouped into several types:

  • Main - describing the top level aspect of the document
  • Metadata - providing information about the page itself
  • Sectioning - organizing the document into logical pieces
  • Text - organizing content
  • Inline Text - define meaning, structure and style of words
  • Multimedia - adding resources to the document
  • Embedded Content - adding other types of resources to the document
  • Scripting - creating dynamic content
  • Demarcating - showing deleted and inserted text
  • Table Content - handling tabular data
  • Form Content - creating forms
  • Interactive Elements - adding interactive user objects
  • Web Components - adding elements to a document

A couple examples of semantic elements

strong makes the text bold

li makes a numered list of items

em creates emphasis
